# Калининград — заметка туриста · Горбилет

Лендинг-гид по Калининграду в фирменной тёмной эстетике Горбилета: hero с Кафедральным собором, секции достопримечательностей, хомлины, активности со скидками, область, сувениры/еда и фотогалерея.

## 📁 Структура
- `index.html` — страница
- `style.css` — стили
- `script.js` — анимации появления при скролле
- `images/` — все фотографии (уже оптимизированы под веб)

## 🚀 Публикация на GitHub Pages
1. Создай репозиторий и загрузи все файлы вместе с папкой `images/`.
2. Settings → Pages → Source → ветка `main`, папка `/root` → Save.
3. Через ~1 минуту сайт будет доступен по адресу `https://<твой-логин>.github.io/<репозиторий>/`.

## 🖼 Как заменить фото
Просто положи новый файл с тем же именем в `images/` (например, `sobor.jpg`). Имена соответствуют содержанию: `sobor`, `homlin`, `amber`, `marzipan`, `forest`, `coast`, `sea`, `swan`, `gates`, `amalienau`, `svetlogorsk` и т.д.
